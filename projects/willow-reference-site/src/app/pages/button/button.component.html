Button Component

The Button component provides a customizable button element with various styles, sizes, and states to fit your application's needs.

Button Variants

Primary Secondary Tertiary Destructive

Button States

Normal State {{isDisabled ? 'Disabled' : 'Enabled'}} Toggle Disabled State

Usage

Import the component in your module and use it in your templates:

 
import { AngularWillowModule } from '@wellmark/angular-willow';

Basic usage:

 
<!-- Basic button -->
<willow-button variant="primary" (click)="onSubmit()">
  Submit
</willow-button>

<!-- Destructive button -->
<willow-button variant="primary" [destructive]="true" (click)="onDelete()">
  Delete
</willow-button>

<!-- Disabled button -->
<willow-button 
  variant="primary" 
  [disabled]="!isValid"
  (click)="onSubmit()">
  Save
</willow-button>

Properties

Property Type Default Description
variant 'primary' | 'secondary' | 'tertiary' 'primary' The visual style variant of the button
htmlType 'button' | 'submit' | 'reset' 'button' The native HTML button type
disabled boolean false Whether the button is disabled
destructive boolean false Whether to apply destructive styling

Events

Event Type Description
click EventEmitter<MouseEvent> Emitted when the button is clicked
