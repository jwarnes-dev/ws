Alert Component

The Alert component provides contextual feedback messages for typical user actions with a handful of available and flexible alert messages.

Basic Example

Usage

Import the component in your module and use it in your templates:

 
import { AngularWillowModule } from '@wellmark/angular-willow';

Basic usage:

 
<willow-alert [messages]="alertMessages" (messageClick)="onMessageClick($event)"></willow-alert>

// Component
alertMessages: ValidationMessage[] = [
  { controlName: 'success', message: 'Success! Your action was completed successfully.' }
];

Properties

Property Type Default Description
messages ValidationMessage[] [] Array of alert messages to display

Events

Event Type Description
messageClick EventEmitter<ValidationMessage> Emitted when a message in the alert is clicked

ValidationMessage Interface

Property Type Description
controlName string The name/identifier for the control or alert type
message string The message text to display in the alert
