Card Component

The Card component provides a simple content container with customizable background colors. Perfect for organizing content in a clean, structured layout.

Usage

Import the component in your module and use it in your templates:

 
import { AngularWillowModule } from '@wellmark/angular-willow';

Basic usage:

 
<!-- Basic card -->
<willow-card>
  <div style="padding: 1.5rem;">
    <h3>Card Title</h3>
    <p>Your card content goes here.</p>
  </div>
</willow-card>

<!-- Card with background color -->
<willow-card [backgroundColor]="'#4A7390'">
  <div style="padding: 1.5rem;">
    <h3 style="color: white;">Colored Card</h3>
    <p style="color: white;">Card with custom background color.</p>
  </div>
</willow-card>

Properties

Property Type Default Description
backgroundColor string undefined The background color of the card (CSS color value)

Content Projection

The card component uses content projection to display any content you place inside it. You are responsible for styling and padding your content appropriately.
Slot Description
Default All content placed inside the card will be projected into the card container
